First-Follow-Set Calculator & LL(1) analyzer

Usage

  • Input your grammar in the provided field. Please note that eps is a reserved keyword representing epsilon.
  • Each rule should be on a separate line, following the format: V -> β, where V is a variable and β is a sequence of variables and terminals including epsilon.
  • The tool will calculate FI(V) to represent the first set of variable V and FO(V) to represent the follow set. Additionally, it will calculate LA(n), the lookahead set for a given rule n.
  • Also, it will analyze if the given grammar is LL(1) or not by intersecting the lookahead-sets with the same variable.
  • An Analysis Table will be created. If the grammar is not LL(1), the regarding cells will be marked red.
  • If you have a left-recursive production rule, the tool will not work.
